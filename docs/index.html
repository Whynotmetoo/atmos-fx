Quick Start

1. Install package via npm:

npm i atmos-fx

2. Import and wrap your app with AtmosFx and AtmosCard:

import { AtmosFx, AtmosCard } from 'atmos-fx'
import 'atmos-fx/styles.css'

export function App() {
  return (
    <AtmosFx preset="rain" density={0.65} wind={-0.12} bottomCollision={true}>
      <div className="layout">
        <AtmosCard transMode="glass" liquidDripping={true}>
          <h1>Interactive Opaque Shelf</h1>
          <p>Precipitation splashes here.</p>
        </AtmosCard>
      </div>
    </AtmosFx>
  )
}

API Reference

AtmosFx Component Config / Props

Option Type / Values Description
preset 'rain' | 'snow' | 'hail' Applies preset default physical and visual values.
particle 'rain' | 'snow' | 'hail' Overrides preset particle rendering without overwriting speed/wind presets.
density number (0 to 1) 0.65 Scales preallocated particle budget and spawn count.
speed number (0 to 10) 1.0 Scalar multiplier for gravity and vertical fall speed.
wind number (-1.5 to 1.5) -0.12 Affects horizontal sway and particle drift.
color string #dcebffb7 CSS color representation for precipitation particles.
quality 'auto' | 'low' | 'medium' | 'high' Limits particle count and device pixel ratio density. Defaults to auto.
bottomCollision boolean false Determines whether particles collide with the bottom edge of the container.
collisionSelector string Query selector for discovering top-edge landing surfaces. Defaults to [data-atmos-collision].
opaqueSelector string Query selector for elements that skip transparency blurs. Defaults to [data-atmos-opaque].

AtmosCard Card Config / Props

Option Type / Values Description
transMode 'glass' | 'opacity' | 'solid' Specifies card transparent integration style. Glass mode triggers high-fidelity backdrop blurs.
liquidDripping boolean true Toggles the water droplets condensation, bulging, and dripping animation (only in Rain mode).
asChild boolean false Merges the AtmosCard properties directly onto the underlying child element instead of rendering a wrapper node.
opacity number (0 to 1) Component-level custom backdrop opacity override (specifically used in opacity mode).

Controller Methods

For direct vanilla JavaScript control, initialize the atmosphere and use the available methods:

import { createAtmosphere } from 'atmos-fx'

const controller = createAtmosphere(document.querySelector('#container'), {
  preset: 'rain',
  density: 0.8,
  bottomCollision: true
})

// Start rendering
controller.start()

// Update parameters dynamically
controller.update({ wind: 0.5, speed: 1.5 })

// Stop and destroy when leaving the page
controller.destroy()
Method Description
start() Initializes canvas layers, starts the requestAnimationFrame loop and rendering.
stop() Stops the animation loop and clears canvases. Does not remove layers.
pause() Pauses the animation execution while keeping the active frame buffer.
resume() Resumes execution from a paused state.
resize() Manually triggers context bounds measurement and canvas resizing.
update(options) Dynamically updates atmosphere parameters on the fly without resetting state.
destroy() Cleans up DOM nodes, listeners, context properties, and cancels loops.
