Options

            

              
                

                  Option
                  Values
                

              
              
                

                  preset
                  
                    rain, storm, snow,
                    hail
                  
                

                

                  density
                  0 to 1; maps into quality-capped budgets
                

                

                  speed
                  Scalar for fall velocity; 0 freezes motion
                

                

                  wind
                  -1 to 1; affects fall angle and drift
                

                

                  quality
                  auto, low, medium, high
                

                

                  transparency
                  glass, opacity, none
                

                

                  surfaceOpacity
                  0 to 1; controls glass surface opacity
                

                

                  contentOpacity
                  0 to 1; controls opacity-mode content fade
                

                

                  snowAccumulation
                  0 to 1; controls snow buildup intensity
                

                

                  collisionSelector
                  Defaults to [data-atoms-collision]
                

              
            

          

        


        

          

            
Performance

            
    
              
  • Use quality: 'auto' for responsive particle budgets.
    • 
              
  • Keep collision surfaces intentional; rects refresh outside the animation loop.
    • 
              
  • Import atoms-fx/styles.css once to enable glass and opacity controls.
    • 
              
  • Leave respectReducedMotion enabled for production UI.
    • 
              
  • Rain uses background and foreground layers so translucent surfaces can reveal rain behind them.
    • 
              
  • Snow accumulation uses a bounded foreground pool and scales with quality and density.
    • 
            

          


          

            
DOM controls

            
    
              
  • data-atoms-collision makes a precipitation landing edge.
    • 
              
  • data-atoms-opaque keeps controls solid.
    • 
              
  • data-atoms-opacity="0.68" sets per-element opacity.
    • 
              
  • data-atoms-glass opts nested surfaces into glass styling.
